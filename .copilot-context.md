## 📝 实施说明

### 对于 Claude Copilot 用户
将此文件保存为 `.claude_context.md` 或 `.clauderc`，系统会在启动时自动加载。

### 对于自定义 AI 系统
在每个新会话开始时执行：
```bash
# 读取进度文件
cat SESSION_PROGRESS_2026-02-11.md

# 读取架构文件
cat .github/copilot-instructions.md

# 显示最新提交
git log --oneline -5
```

### 触发条件
- 用户输入: `keep go`
- 用户输入: `继续`
- 用户输入: `load context`
- 新会话启动时自动加载 (可选)
